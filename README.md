Contains resources for the pyISSM workshop held at Monash University, March 2026. 

Files included: 

  - pyISSM_installation_guide.ipynb      Guide for installing and loading pyISSM and the ACCESS Cryosphere Data Pool on the cluster Gadi using ARE.

  - pyISSM_meshing_guide.ipynb           Guide for creating first ice sheet model mesh using pyISSM and the model object.
  
  - pyISSM_parameterisation_guide.ipynb  Guide for parameterising the model object. Note, not all relevant fields are covered here.
  
  - PineIslandParam.py                   Parameterisation file containing commands to complete model parameterisation. Dependency of pyISSM_parameterisation_guide.ipynb.
  
  - newWilkesDomain.exp                  Exp file for outlining the Wilkes Subglacial Domain mesh. Dependency of pyISSM_meshing_guide.ipynb.
